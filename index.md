# Welcome to your Career Coach

This is a fresh instance — nothing is set up yet.

**To begin, type `/setup` in Claude Code.** It will:

- install dependencies,
- read your résumé,
- ask a few questions to calibrate your search (writes `USER.md`),
- build your first dashboard, and
- optionally create a private backup repo.

Once set up, this page becomes the catalog of every page in your wiki.

New here? Open `README.md` for the full overview, or look at `examples/` to see a populated instance.
